This special-purpose application enables running custom boot code on Tegra X1 devices such as the Nintendo Switch.

Batteries are not included: This application contains relocators and {re}switched's demo binaries, but you'll have to import custom payloads with actual functionality yourself.

Warning and Disclaimer

Early boot code has full access to the device it runs on and can damage it. No such code is shipped with this application. Responsibility of executing boot code remains with the user.

Hardware Requirements

An adapter is required for connecting your iOS device to the target Tegra X1 device. The following are known to work:

  • USB-C to Lightning adapters with data lines (charging only adapters do not work)
  • USB-C to Lightning cables
  • Apple Lightning USB 3 Camera Adapter with a USB 3 to USB 3 Type C cable

Importing Boot Code

Use Safari or any cloud storage provider, including iCloud Drive, to make Boot Code available in iOS' files app. Afterwards the Boot Code may be added to a Boot Profile.

Create Boot Profiles by opening the Boot Code List from thie app's start page and selecting New Profile.

A description and two files must be provided:

The first file is the relocator binary. The relocator is responsible for cleaning up the boot session and preparing the second file for execution. Two relocators are shipped with this app: cbfs.bin (for loading Coreboot/Linux) and intermezzo.bin (for loading any custom firmware payload.) You may also use a custom relocator.

The second file is the payload. For Coreboot/Linux this file is usually named coreboot.rom. Other well-known payloads are fusee.bin, SX OS's payload.bin or the Hekate bootloader's hekate_ctcaer_X.Y.bin.

Selected relocator and payload files are copied to this application's storage, so that they don't have to remain in iOS' Files or cloud storage.

Booting the Target Device

  1. Select your Boot Code on this application's start page after importing it.
  2. In any order:
    • Enter RCM mode on the target device.
    • Connect the target device via USB. A warning about an unsupported accessory appears and can be ignored.
    • Tap this app's Boot Now button.
  3. All detected Tegra X1 devices in RCM mode are automatically provisioned with the selected Boot Code. This app must remain in foreground for automatic booting.

Tips and Tricks

The following section applies to the Nintendo Switch:

  • Enter RCM mode by 1. shorting the home pin to a ground pin on the right Joy-Con rail, 2. holding down Volume +, 3. tapping the POWER button once.
  • In RCM the screen remains black, but the device is detected as 'APX' by PCs and will work with this app.
  • Exit RCM mode without booting custom code by holding the POWER button for 12 seconds.
